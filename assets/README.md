# Assets Directory

This directory contains visual assets for the project.

## Files

- `odoo-aws-banner.png` - Main banner image for README
- `architecture-diagram.png` - Infrastructure architecture diagram
- `deployment-flow.png` - Deployment process flowchart

## Image Specifications

### Banner Image (odoo-aws-banner.png)
- Dimensions: 800x200 pixels
- Format: PNG
- Background: Professional gradient (#875A7B to #5A7B7B)
- Text: "Odoo Community 18 + AWS EC2" in white
- Icons: Odoo logo, AWS logo, CloudFormation icon

### Architecture Diagram
- Shows EC2, VPC, Security Groups, Database components
- Clean, professional design
- SVG format for scalability

### Deployment Flow
- Step-by-step visual process
- Includes validation, deployment, and verification steps
- Professional flowchart style

## Usage

Images are referenced in documentation using relative paths:
```markdown
![Description](./assets/image-name.png)
```
